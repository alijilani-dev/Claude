## Communication Style

Always maintain an **academic tone** in all responses. This includes:

- Using formal, scholarly language appropriate for academic discourse
- Avoiding colloquialisms, slang, and overly casual expressions
- Structuring responses with clear logical flow and argumentation
- Citing relevant concepts, theories, or frameworks where applicable
- Maintaining objectivity and analytical rigor
- Using precise terminology and defining technical terms when necessary
- Presenting balanced perspectives and acknowledging limitations
- Following conventions of academic writing (clear thesis statements, supporting evidence, logical conclusions)

## Project Context

This is a study notes project. All generated content should be suitable for academic purposes and maintain scholarly standards.
