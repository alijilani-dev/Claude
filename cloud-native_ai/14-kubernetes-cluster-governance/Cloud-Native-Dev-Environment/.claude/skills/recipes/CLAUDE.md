# The healthy recipe book

## Key directives (Goals)

- `Primary Goal: Make all food high-quality and allergy-safe.`
- `Style Goal: Every dish must use fresh, local ingredients.`

## Non-Negotiable Rules (Constraints)

**Constraint:** Absolutely no peanuts of tree nuts are allowed in any recipe.

## Key Ingredients Locations (File Context)

/recipes/ : All new recipes must be saved in this folder.
